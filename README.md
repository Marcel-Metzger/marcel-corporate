# Marcel-Philippe Metzger — Corporate Landing Page

A single-page "Corporate Professional" landing page for Marcel-Philippe Metzger,
focused on quantitative finance & risk. Calm visual hierarchy, generous spacing,
a deep trust-blue palette with warm neutrals and a single amber accent.

## Stack
- Static HTML / CSS / vanilla JavaScript — no build step required.
- Fonts: Source Serif 4 (headings) + Inter (body), via Google Fonts.

## Structure
- `index.html` — page markup and content.
- `style.css` — design system, layout and responsive rules.
- `script.js` — scroll progress, reveal-on-scroll, animated counters, active-nav.

## Run locally
Open `index.html` in a browser, or serve it:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
