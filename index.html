Financial Data Pipeline

End-to-end system for automated collection, processing and storage of financial market data designed for ML models and derivatives valuation. Features GARCH(1,1) volatility forecasting, cross-validation between data sources and intelligent missing-value handling using Kalman filtering, with multi-layer outlier detection combining statistical methods and Isolation Forest.

Trading News Aggregator

Multi-source news aggregation system for financial markets with automated ticker extraction and semantic story clustering.
